# Borderlands4-Mod

## 概览
本模组通过 DLL 入口 Hook、自定义界面和辅助工具扩展《Borderlands 4》，包含引擎拦截、ImGui 界面、配置文件与最小依赖（例如 minhook）。

## 依赖
- Windows 10/11 系统
- Visual Studio 2022 或兼容的 MSVC 工具集
- 与当前游戏版本保持兼容（每次更新后务必重新确认）

## 构建与部署
1. 在 Visual Studio 中打开 `Borderlands4.sln`，选择 `x64 Release` 配置并生成。
2. 将生成的 DLL（通常位于 `x64/Release`）复制到游戏根目录。
3. 通过注入器/启动器启动游戏，使其加载该 DLL，然后根据 `Config` 目录下的文件调整行为。

## 使用说明
- 修改配置文件后重启游戏以生效。
- 按设定热键呼出 ImGui 界面，直接调整辅助功能或查看状态。
- 依赖 `minhook`、`Hooks`、`Utils` 等模块完成内存拦截和数据交互。

## 安全与合规
- **开启此模组时不得加入任何公共房间。**仅在私人或单人环境中使用，避免影响其他玩家。
- 使用前务必理解本模组的行为并接受潜在的账号/平台风险。
- 避免同时运行多个注入器，以防冲突或触发检测。
